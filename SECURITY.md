# Security Policy

## Supported Versions

The **Manas AGI Project** is currently in an active, experimental research phase. 

| Version | Supported          |
| ------- | ------------------ |
| Main    | :white_check_mark: |
| Legacy  | :x:                |

*(Note: As this is a research-focused Artificial General Intelligence project, it is not currently recommended for production deployments where strict security guarantees are required.)*

## Reporting a Vulnerability

Security is a priority. If you discover a security vulnerability within Manas, please do not disclose it publicly. 

Instead, please report it privately:
1. Contact the project owner (Zeeshan Rahman) directly.
2. Or use GitHub's private vulnerability reporting feature on this repository.

We will review all reports and address significant security flaws as quickly as possible. Please allow reasonable time for us to respond before making any information public.
