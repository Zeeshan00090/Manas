# Manas AGI

Welcome to the **Manas** project. Manas is an ambitious research initiative aimed at building a true Artificial General Intelligence (AGI) by moving beyond the limitations of standard Transformer architectures and Next-Token Prediction (NTP).

## Core Vision: Meaning Before Language
Current LLMs operate as stochastic parrots—predicting the next word based on statistical likelihood. Manas introduces the **Semantic State Evolution (SSE)** paradigm. 

The core philosophy is that a true intelligence must **"think first, and speak later."**
Instead of processing language token-by-token to guess the next word, Manas:
1. **Perceives** language and extracts pure meaning into a high-dimensional Semantic State Vector (SSV).
2. **Reasons** by evolving this state continuously over time using novel mathematics (ZULIC Operators and Navier-Stokes fluid dynamics).
3. **Articulates** by translating the final, resolved thought state back into human language.

## Key Components (Current & Experimental)
- **`Manas.py`**: The core brain containing the GoalTree, Hierarchical Planner, and experimental Liquid ZULIC networks.
- **`nexus.py`**: The orchestrator that binds the cognitive engines with sandbox execution environments and external tools.
- **`understanding_extractor.py`**: A system designed to map and extract causal logic and deep reasoning from data, rather than just surface-level text patterns.

## The Mathematical Foundation
Manas replaces standard Backpropagation and Cross-Entropy loss with:
- **Joint Embedding Predictive Architecture (JEPA)** inspired loss functions (Semantic Contrastive Loss).
- **ZULIC Operators**: Custom non-linear mathematical transformations designed to model cognitive leaps.

## Status
Manas is currently in deep active research and architectural transition. We are deprecating legacy Transformer blocks (`zulic_mlx.py`) in favor of pure Semantic State Autoencoders and Continuous-Time Cognitive Engines.
